

Cahier de recette - Plan de test End-to-End du parcours visiteur 

 

N.B. : Le nombre de scénarios n’est pas déterminé. On peut en ajouter ou en supprimer si besoin.  

 

 

Scénario 1 : Filtrage des événements par catégorie 

 

Given 

Je consulte la liste des événements. 

When 

Je sélectionne une catégorie via le sélecteur. 

Then 

La liste des événements affiche uniquement les événements de la catégorie sélectionnée. 

 

 

Scénario 2 : Affichage d’une carte événement 

 

Given 

Je consulte la liste des événements. 

When 

Un événement est affiché. 

Then 

La carte affiche l’image, le titre, le label (type) et le mois de l’événement. 

 

 

Scénario 3 : Affichage d’une carte membre de l’équipe 

 

Given 

Je consulte la liste des membres de l’équipe. 

When 

Un membre est affiché. 

 

 

Then 

La carte affiche l’image, le nom et le poste du membre. 

 

 

 

 

 

Scénario 4 : Utilisation du bouton 

 

Given 

Un bouton est affiché sur la page (ex : “Contact”). 

When 

Je clique sur le bouton. 

Then 

L’action définie dans la prop on Click est exécutée (ex : navigation vers la section contact). 

 

 

Scénario 5 : Utilisation du champ de formulaire 

 

Given 

Un champ de formulaire est affiché (ex : nom, email, message). 

When 

Je saisis du texte dans le champ. 

Then 

La valeur saisie est prise en compte et la fonction onChange est appelée. 

 

 

                 Scénario 5 : Sélecteur déroulant dans le formulaire 

 

Given 

Le formulaire de contact est affiché. 

When 

Je clique sur le sélecteur “Personnel / Entreprise” et je choisis une option. 

 

 

Then 

L’option sélectionnée est affichée dans le formulaire et prise en compte lors de la soumission. 

 

 

 

 

 

 

 

 

 

Scénario 6 : Ouverture et fermeture d’un modal événement 

 

Given 

  Je consulte la liste des événements. 

When 

Je clique sur une carte événement. 

 

 

Then 

Une fenêtre modale s’ouvre et affiche les détails de l’événement (image, titre, période, description, participants, prestations). 
Je peux fermer la fenêtre modale en cliquant sur la croix. 

 

 

Scénario 7 : Pagination de la liste des événements 

 

Given 

La liste des événements contient plus de 9 éléments. 

When 

Je clique sur un numéro de page dans la pagination. 

Then 

La liste affiche les événements correspondant à la page sélectionnée. 

 

 

 

Scénario 10 :  Envoi du formulaire de contact 

 

Given 

Le formulaire de contact est affiché et rempli. 

When 

Je clique sur le bouton “Envoyer”. 

Then 

Le formulaire est envoyé, le bouton affiche “En cours”, puis “Envoyer” à nouveau une fois l’envoi terminé. 
Le callback on Success est appelé. 

 

 

 

 

 

 

Scénario 11 : Slider des événements à la une 

 

Given 

Je suis sur la page d’accueil. 

When 

Le slider d’événements est affiché. 

Then 

Les événements défilent automatiquement toutes les 5 secondes, et je peux voir le titre, la description et le mois de chaque événement. 

 

 

Scénario 12 : Accessibilité clavier du menu de navigation 

 

 

Given 

Je suis sur la page d’accueil. 

When 

J’utilise la touche “Tab” pour naviguer dans le menu principal. 

Then 

Chaque lien du menu (“Nos services”, “Nos réalisations”, “Notre équipe”, “Contact”) est accessible et peut être activé avec la touche “Entrée”.
